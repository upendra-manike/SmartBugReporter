# Privacy policy URL for Chrome Web Store

The Chrome Web Store requires a **publicly accessible privacy policy URL**.

## Option 1: GitHub Pages (recommended)

1. Put **`privacy-policy.html`** in your repo (e.g. in a `docs/` folder or repo root).
2. In the repo: **Settings → Pages**.
3. **Source:** Deploy from a **branch** (e.g. `main`), folder **/ (root)** or **/docs**.
4. After the site builds, your URL will be:
   - `https://<YOUR_USERNAME>.github.io/<REPO>/privacy-policy.html`
   - or `https://<YOUR_USERNAME>.github.io/<REPO>/` if you use `index.html` for the policy.

**Example:** Repo `myuser/SmartBugReporter` →  
`https://myuser.github.io/SmartBugReporter/privacy-policy.html`

## Option 2: Your own website

Upload **`privacy-policy.html`** to your site and use that URL (e.g. `https://yourdomain.com/smart-bug-reporter-privacy`).

## What to enter in the Chrome Web Store

In **Privacy practices** → **Privacy policy**, paste the full URL where the policy is publicly visible.
